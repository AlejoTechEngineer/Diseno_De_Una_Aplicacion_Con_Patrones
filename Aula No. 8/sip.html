¿Qué es SIP?

SIP (Session Initiation Protocol) es un protocolo de señalización que permite iniciar, mantener y finalizar sesiones multimedia entre dos o más participantes en una red IP.

Opera sobre TCP/UDP, generalmente en el puerto 5060, y utiliza mensajes de texto similares a HTTP.

Componentes del Protocolo SIP

  • User Agent (UA): Usuario que envía o recibe la comunicación.
  • Proxy Server: Encargado de enrutar solicitudes.
  • Registrar: Registra la localización del usuario.
  • Location Service: Asiste en la localización del destino.

Establecimiento de Sesión SIP

La sesión se establece con un flujo de mensajes como el siguiente: 

INVITE →
 100 Trying ←
180 Ringing ←
 200 OK ←
    ACK →
